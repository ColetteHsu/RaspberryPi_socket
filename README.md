## RaspberryPi_socket

功能介紹：在一開始透過「UDP/TCP/Rest Network Test Utility」連線樹莓派的ip位置，完成手機控制樹莓派開關燈功能。 
1.ron - 開紅燈 
2.yon - 開黃燈 
3.gon - 開綠燈 
4.allon - 三顆燈都開 
5.roff - 關紅燈 
6.yoff - 關黃燈 
7.goff - 關綠燈 
8.alloff - 三顆燈都關。
